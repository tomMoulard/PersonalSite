Question R13

Suppose users share a 2 Mbps link. Also suppose each user transmits continuously at 1 Mbps when transmitting, but each user transmits only 20 percent of the time. (See the discussion of statistical multiplexing in Section 1.3.)

a. When circuit switching is used, how many users can be supported?
b. For the remainder of this problem, suppose packet switching is used. Why will there be essentially no queuing delay before the link if two or fewer users transmit at the same time? Why will there be a queuing delay if three users transmit at the same time?
c. Find the probability that a given user is transmitting.
d. Suppose now there are three users. Find the probability that at any given time, all three users are transmitting simultaneously. Find the fraction of time during which the queue grows.

Response R13

a. If each user uses 1 Mbps when it is active, only two user can use the connection simultaneously since they share a 2 Mbps link.
b. If there is only two or fewer user using the connection at the same time, there will be no waiting time ether way. But by using packet switching, the connection (2 Mbps) will be spliced into users (1Mbps each) and this way, the queue won't be used.
c. The probability of a packet from a specified user is going threw the link is simply by dividing one by the number of current user.
d. The probability is 0.008 or 0.8%.

Question R23

What are the five layers in the Internet protocol stack? What are the principal responsibilities of each of these layers?

Response R23

Theses are the different layers of the Internet Protocol Stack :

The Application Layer  This layer create the message or use it.
The Transport Layer This layer decide which protocol to use for the next layer. UPD is faster but less secure and TCP is slower but far more secure.
The Network Layer This layer transform the message into packet using different protocols like UDP and TCP.
The Link layer This layer add the last header that specifies the route to take to deliver the message.
The Physical Layer This one is the layer that takes care of transmitting the message to the correct receiver thanks to headers added by previous layers.
